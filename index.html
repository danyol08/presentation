The Challenge
Barcotech Philippines, Inc.
Managing physical assets across multiple sites is hard.
⚠️
Spreadsheet chaos — tracking assets manually is error-prone, slow, and impossible to scale.
📍
Zero real-time visibility — no way to know where an asset is or what condition it's in right now.
🔄
No structured workflow — deploy and pull-out events are undocumented and inconsistent.
📋
Missing audit trail — when an asset changes hands, there's no record of who, when, or why.
📊
Reporting is painful — generating any report requires hours of manual data collection.

Scan & Asset Lookup
Barcotech Philippines, Inc.
Fixed Asset Manager / Scan

Find Any Asset Instantly

Type or scan an asset code — QR and barcode compatible
Instant lookup returns: name, condition, status, and current location
From results: Deploy or Pull Out with one tap
Works offline using local SQLite cache — no internet required in the field
Designed for field users on Windows tablets or desktops
Condition Tagging on Deploy / Pull-Out
Barcotech Philippines, Inc.
Fixed Asset Manager / Condition Tag

Log Condition at Every Movement

Triggered on every Deploy or Pull-Out event — never skipped
Select condition: Good  Fair  Damaged
Choose deployment site from the managed location list
Add optional notes for context or damage details
Every tag is timestamped and synced to Supabase automatically
Builds a complete condition history per asset over its lifecycle
